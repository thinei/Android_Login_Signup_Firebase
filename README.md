# Android_Login_Signup_Firebase
Authentication Using Firebase


###### There are four Activities.
          1.MainActivity.java(main)
          2.Registration.java
          3.LoginActivity
          4.DashboardActivity.java
    
###### It comprises of four layouts respectively.
          1.activity_main.xml
          2.activity_registration.xml
          3.activity_login.xml
          4.activity_dashboard.xml
**********************************************************************************

###### Steps to run the app.
          1.Signup using the asked details and full fill all the requirements.
          2.After signup process success, can logged-in to system.


***********************************************************************************

###### To run the app on your own database,follow the steps.
          1. Add new project to the firebase.
          2. Download the "google-services.json".
          3. Put this "google-services.json" file under ....app/  directory (eg. app/google-services.json).
          4. Edit config in firebase to enable the Authentication through email and password in setup method tab.
          5. Now the database has been changed to your own database successfully.
